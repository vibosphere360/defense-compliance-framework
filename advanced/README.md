# Advanced Compliance Modules

This directory contains advanced scenarios, integrations, and controls beyond the core 120-minute training program.

## Modules
- `incident-response/` — Playbooks for ITAR/CUI breach scenarios
- `advanced-policies/` — Complex threat scenarios (insider threat, supply chain)
- `grc-integrations/` — Examples for Archer, ServiceNow, API evidence collection
- `cmmc-level-3/` — Enhanced controls for CMMC Level 3 / NIST 800-172
- `ai-governance-advanced/` — Advanced AI/ML risk scenarios and enforcement

> These modules assume familiarity with the core training program. Use for senior compliance staff, engineers, and leadership briefings.
